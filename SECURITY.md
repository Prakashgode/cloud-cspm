# Security Policy

## Supported Version

This project is pre-1.0. The supported version is the latest code on `master`.

## Reporting a Vulnerability

Please do not open a public GitHub issue for a suspected vulnerability.

Report security issues privately to the maintainer at `gode.prakash@gmail.com`
with:

- a short summary of the issue
- affected version or commit
- steps to reproduce
- impact and any suggested mitigation

I will acknowledge the report as soon as practical and work with you on
validation, remediation, and coordinated disclosure.
